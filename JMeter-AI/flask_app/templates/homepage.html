About PWPM

PWPM is a powerful platform for conducting load testing using JMeter. It analyzes test results and leverages an LSTM model for anomaly detection, providing actionable insights. The results are visualized and summarized, helping you understand system behavior under stress.

Features

Data Analysis & Visualization

Test data is visualized for better understanding of system behavior.

AI-Driven Anomaly Detection

Leverage the power of LSTM models to detect anomalies in system performance.

Dynamic Test Data

Run load tests with dynamic and updated test data for accurate results.
